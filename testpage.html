THE GAME OF CHAOS

OR....

The chaos metod to get The Sierpinski triangle

Whats happend ?

In simple words - we take three random points on the plane (A,B,C).Then we take one more in random place (start point). Further we need to get a random number. We will use a "dice roll". Every roll if we get 1 or 2 we will make new start point between old start point and A point. If we get 3 or 4 we will make new start point between old and B points. And if we get 5 or 6 we will make point between old start point and C.

This is rules the game of chaos. When I read it at first time I thought that if we make over 10000 iterations then all place inside triangle ABC will be fill.

In this momment try to make 5 , 10, 50, 100 and 1000 iterations and look on the graph !!!

You see The Sierpinski triangle - one of the types of fractals.
